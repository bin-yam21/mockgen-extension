# MockGen VS Code Extension

## Features
- Scan your workspace JS/TS files for API endpoints
- Generate realistic mocks (`.mockgen/mock.json`)
- Start a local Mock server
- View Swagger/OpenAPI UI and dashboard
- Inline mock editing & charts

## Commands
| Command | Description |
|---------|-------------|
| MockGen: Scan API Endpoints | Scan project for endpoints |
| MockGen: Generate Mock API | Create `.mockgen/mock.json` |
| MockGen: Start Mock Server | Start local mock server on port 3000 |
| MockGen: Reload Mock Server | Reload mocks without restarting |
| MockGen: View Dashboard | Open Dashboard with charts & Swagger |

## Usage
1. Open a workspace folder
2. Run `MockGen: Scan API Endpoints`
3. Generate mocks
4. Start server and view Swagger/UI
